# nancythanki.com

Nancy Thanki's personal website built with Jekyll.

## Setup

To run this Jekyll site locally, you'll need Ruby >= 2.7. Then:

```bash
bundle install
bundle exec jekyll serve
```

Visit http://localhost:4000 to see your site.

## Structure

- `_config.yml` - Site configuration
- `index.md` - Homepage
- `about.md` - About page
- `_posts/` - Blog posts directory (create as needed)
